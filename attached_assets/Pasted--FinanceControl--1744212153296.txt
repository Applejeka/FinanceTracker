ехническое задание на разработку приложения "FinanceControl" для анализа и оптимизации личных расходов
1. Введение
1.1 Наименование программного продукта
"FinanceControl" - приложение для учета и анализа личных финансов.
1.2 Назначение разработки
Создание удобного инструмента для контроля доходов и расходов с возможностью визуа-лизации данных и генерации отчетов.
2. Требования к функциональным характеристикам
2.1 Основные функции:
•	Учет доходов и расходов
•	Категоризация финансовых операций
•	Визуализация данных (графики, диаграммы)
•	Генерация отчетов
•	Управление транзакциями (добавление, редактирование, удаление)
2.2 Требования к интерфейсу:
•	Главное окно с навигационным меню
•	Форма добавления транзакций
•	Просмотр истории операций
•	Раздел с аналитикой и визуализацией данных
•	Настройки приложения
3. Технические требования
3.1 Используемые технологии:
•	Язык программирования: Python 3.10+
•	Графическая библиотека: PyQt6
•	База данных: SQLite3
•	Дополнительные библиотеки: matplotlib, pandas
3.2 Системные требования:
•	Операционная система: Windows 10/11
•	Процессор: 2 ядра, 1.6 GHz
•	Оперативная память: 4 GB
•	Дисковое пространство: 50 MB
4. Описание модулей приложения
4.1 Основные модули:
1.	main.py - точка входа в приложение
2.	database.py - работа с базой данных SQLite
3.	models.py - модели данных (транзакции, категории)
4.	views.py - графический интерфейс (PyQt)
5.	reports.py - генерация отчетов и визуализация данных
6.	settings.py - настройки приложения
5. Требования к базе данных
5.1 Структура БД:
1.	Таблица transactions:
o	id (INTEGER, PRIMARY KEY)
o	amount (REAL)
o	category_id (INTEGER, FOREIGN KEY)
o	date (TEXT)
o	description (TEXT)
o	type (TEXT) - income/expense
2.	Таблица categories:
o	id (INTEGER, PRIMARY KEY)
o	name (TEXT)
o	color (TEXT)
6. Графический интерфейс
6.1 Основные окна:
1.	Главное окно:
o	Меню навигации
o	Краткая статистика
o	Быстрый доступ к основным функциям
2.	Окно добавления транзакции:
o	Поля: сумма, категория, дата, описание
o	Кнопки: сохранить, отменить
3.	Окно истории транзакций:
o	Таблица с операциями
o	Фильтры по дате и категориям
o	Кнопки: редактировать, удалить
4.	Окно аналитики:
o	Диаграммы расходов по категориям
o	Графики динамики доходов/расходов
o	Кнопка генерации отчета
